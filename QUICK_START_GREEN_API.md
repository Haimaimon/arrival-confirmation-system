# ⚡ Quick Start - Green API (5 דקות)

## 🚀 הגדרה מהירה

### **שלב 1: הרשמה (2 דקות)**
```
1. פתח: https://green-api.com
2. לחץ: Sign Up / הרשמה
3. מלא פרטים ואשר אימייל
```

### **שלב 2: יצירת Instance (1 דקה)**
```
1. בלוח הבקרה: Create Instance
2. העתק את:
   - Instance ID: 1101234567
   - API Token: abc123def456...
```

### **שלב 3: חיבור WhatsApp (1 דקה)**
```
1. בלוח הבקרה: Link Device
2. WhatsApp בטלפון → הגדרות → Linked Devices
3. סרוק את ה-QR Code
4. מחובר! ✅
```

### **שלב 4: הוספת Credentials למערכת (1 דקה)**

**צור/ערוך קובץ:** `packages/backend/.env`

```env
# הוסף את השורות הבאות:
GREEN_API_INSTANCE_ID=1101234567
GREEN_API_TOKEN=abc123def456ghi789jkl012mno345pqr678
```

**החלף את הערכים עם אלו שקיבלת מ-Green API!**

### **שלב 5: אתחול מחדש**
```bash
# עצור את השרת (Ctrl+C) והפעל שוב:
cd packages/backend
npm run dev

# חפש בלוג:
✅ Green API configured successfully
```
